About The Watch Gallery

The Watch Gallery is a curated online boutique for wrist watches — classic, sporty, luxury and smart designs. This demo site is a front-end only project built with pure HTML, CSS and JavaScript and uses browser localStorage to simulate a backend.

What you can do here

  • Browse products and search by name
  • Register / Login (users & admin)
  • Add to cart and complete a mock payment
  • Admin can add/remove products and manage orders

This demo uses Unsplash images and is intended for local/testing use.

Order History

When logged in, you can view your order history on the Cart page. Orders are stored locally in your browser.
